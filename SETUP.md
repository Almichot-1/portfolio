# Quick Setup Guide

## Prerequisites

Install Node.js 20+ from: https://nodejs.org/

## Installation

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

1. Push to GitHub
2. Import project on vercel.com
3. Deploy (auto-detects Next.js)

Done. That's it.
