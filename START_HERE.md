# START HERE

## You Need Node.js First

Install from: **https://nodejs.org/** (get version 20 or higher)

## Then Run These 3 Commands

```bash
npm install
npm run dev
```

Open **http://localhost:3000**

## Deploy

Push to GitHub, then import on **vercel.com**

That's it. Everything else is ready.

## Customize

Before deploying, update:
1. Email in `components/sections/hero.tsx` (line 42)
2. GitHub links in `components/sections/systems.tsx` (lines 67, 108, 149)
3. Contact info in `components/sections/contact.tsx` (lines 40-60)

Done.
